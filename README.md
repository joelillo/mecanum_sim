# mecanum_sim
Simple  mecanum robot simulation

## System Requirements:
* Ubuntu 16.04
+ ROS Kinetic
+ Gazebo 7

## Installation:
Uses catkin build system: http://wiki.ros.org/catkin

* Clone this repo into your catkin workspace
+ catkin_make
+ source devel/setup.bash

## Run

* source devel/setup.bash
+ roslaunch mecanum_sim mecanum.launch
